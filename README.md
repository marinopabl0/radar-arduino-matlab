------------------------------------------------
                         RADAR
------------------------------------------------
En este repositorio se encuentra el código fuente:
  radar_1.ino: Es el código que se sube a la placa arduino
  radar_1.m: Es el código que se ejecuta en matlab

En el radar se utilizaron los siguientes:
HADWARE:
  -Placa Arduino
  -Servo Motor
  -Ultrasonico
  -Protoboard
  -Cables
SOFWARE:
   -IDE Arduino
   -Matlab
